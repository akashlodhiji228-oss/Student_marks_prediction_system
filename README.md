# Student Marks Prediction using Linear Regression

This project demonstrates a simple Machine Learning regression model using Python and Scikit-learn to predict student marks based on study hours.

## 🚀 Features

* Load dataset using Pandas
* Train a Linear Regression model
* Predict student marks
* Evaluate model performance using:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * Root Mean Squared Error (RMSE)

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 📂 Dataset

The dataset contains:

* `time_study` → Number of study hours
* `Marks` → Student marks

## ▶️ How to Run

1. Install required libraries:

```bash
pip install pandas numpy scikit-learn
```

2. Run the Python file:

```bash
python filename.py
```

3. Enter study hours when prompted.

